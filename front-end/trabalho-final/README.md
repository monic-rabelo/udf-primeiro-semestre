## Projetos front-end - CSS, HTML e JavaScript

### Projeto02
Template de página Parallax, construído com o framework Materialize. É uma landing page que aplica os conteúdos passados sobre o framework.

* Framework: Materialize CSS.
* Design: Utiliza o efeito parallax (rolagem de fundo em velocidade diferente do conteúdo) para criar profundidade.
* Estrutura: Contém uma barra de navegação responsiva (com sidenav para telas menores), um grande banner de destaque, uma seção de ícones/características (flash_on, group, settings), uma seção "Sobre Nós" e um rodapé completo.
* Materialize: Framework CSS/UI.

### provaA1Grupo07
Este projeto é um site de informações/e-commerce focado em Caminhões de Mineração, utilizando o framework Bootstrap. Ele possui uma navegação para modelos específicos e integra um mapa interativo.

* Framework: Bootstrap (versão 5+ pelas classes g-5, mb-4, etc.).
* Tema: Website para uma "loja" ou portal de informações sobre Caminhões de Mineração.
* Navegação: Links principais para modelos de caminhão específicos (Terex Titan, Caterpillar 797B, Liebherr T282B).
* Recursos Visuais: Integração com Font Awesome e Bootstrap Icons para ícones, e uma seção de mapa interativo (utilizando a biblioteca Leaflet).

---

### Tecnologias Utilizadas
* HTML/CSS/JavaScript: Base do projeto.
* Bootstrap: Framework CSS/UI.
* Leaflet: Biblioteca JavaScript para mapas interativos.
* Fontes: Font Awesome, Bootstrap Icons, e Google Fonts (Quicksand).
